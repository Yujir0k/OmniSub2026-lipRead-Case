# Pretrained VSR + LM + MediaPipe (ориентир верхней планки)

Этот раздел хранит результаты исследовательского этапа, где предобученные модели использовались **только как ориентир** для понимания сильных подходов в lipreading.

## Цель этого этапа

- оценить, какой эффект дают сильный visual backbone + language model + tuned decode;
- сравнить верхнюю планку с нашим финальным from-scratch контуром;
- перенести в финальную систему только безопасные инженерные идеи (например, decode/postprocess), не нарушая правила.

## Что важно по правилам

Материалы в этой папке не являются финальным submission-контуром.  
Финальный проект запускается из корня репозитория (`TinyLipCTC + beam + wordnorm`) и обучается с нуля.

## Результаты ориентира

Основная сводка:

- `metrics_final.json`

Ключевые значения:

- `baseline_original_decode_wer`: `0.22141119221411193`
- `baseline_tuned_decode_wer`: `0.1897810218978102`
- `finetuned_tuned_decode_wer`: `0.1873479318734793`

## Вывод

Этап подтвердил, что предобученные VSR-решения с LM и decode tuning действительно дают более низкий WER, но из-за лицензионных ограничений такие подходы в этой работе использовались только как исследовательский ориентир, а не как финальный путь отправки.
